# Golang application to get messages from a rabbitmq

### requirements:
- rabbitmq
- golang

### building a golang application
```
 GOOS=linux GOARCH=amd64 go build -o go-receive go-receive.go
```

### running this applicatino
```
/app/go-receive
```
